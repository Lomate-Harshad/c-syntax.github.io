About the Project

This web-based platform was developed as a requirement for the Web Page Development (WPD) course. The objective of the project is to provide a digital interface for the dissemination of information related to the Programming in C (PIC) curriculum under the MSBTE K-Scheme.

Development Objectives

The project was executed by three students from the Computer Engineering department. The development process involved the application of standard web technologies to create a functional and navigable layout. The primary goal was the successful completion of the academic criteria associated with the final semester submission.

Team Composition

  • Harshad Lomate Technical implementation and layout structure
  • Om Kadam User interface formatting and styling
  • Govardhan Gurale Documentation and content organization

Technical Specifications

The site is built using HTML and CSS frameworks to ensure compatibility across standard browser environments. All code examples provided within the articles have been formatted to meet common C programming standards.
