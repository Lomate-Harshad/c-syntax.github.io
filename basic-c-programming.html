Basics of C Programming

A practical look at the C syntax and logic

Harshad Lomate

Team Member

For every Computer Engineering student under the MSBTE K-Scheme, the Programming in C (PIC) course is the gateway to core computing. C is a robust, middle-level language that balances the ease of high-level languages with the power of low-level hardware interaction.

The Anatomy of a C Program

Every C program follows a standard structure required for compilation. From the preprocessor directives to the main execution block, understanding this hierarchy is essential for error-free coding.

// Documentation: Basic Hello World program #include <stdio.h> int main() { // Output function from stdio.h printf("Welcome to MSBTE Diploma!\n"); return 0; }

Data Types and Variables

Variables are named memory locations. In C, we categorize data to optimize memory usage. The standard types you will encounter in your practicals include:

  • int: Used for whole numbers (2 bytes in 16-bit, 4 bytes in 32-bit).
  • float: Used for single-precision decimal values.
  • char: Stores a single character (1 byte).

Input and Output Functions

Interaction with the user is handled via printf() and scanf(). These are defined in the Standard Input-Output header file.

int roll_no; printf("Enter your Roll Number: "); // %d is the format specifier for integers scanf("%d", &roll_no);

Arithmetic Operators

C provides a comprehensive set of operators to perform calculations. Along with standard addition and multiplication, the % (modulus) operator is frequently used in MSBTE logic programs to find remainders or check for even/odd numbers.
